# OpLæs

Dansk tekst-til-tale desktop-app til macOS.

- Bygget med Rust + Tauri
- Bruger Plapre / Plapre Nano modellerne
- Konverterer TXT, MD, PDF til tale

## Download

- [Seneste release](https://github.com/RJuro/oplaes/releases/latest)

## Installation

1. Download DMG fra Releases.
2. Dobbeltklik for at mounte.
3. Træk `OpLæs.app` til `Programmer`.

Hvis macOS advarer ved første start: Højreklik appen og vælg `Åbn`.
